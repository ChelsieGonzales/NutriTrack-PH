# NutriTrack PH

NutriTrack PH is a web-based application designed to help barangays and local health units monitor and manage child nutrition data efficiently. It allows tracking of children's height, weight, BMI, and nutritional status, with dynamic charts and tables for easy visualization.

## Features

- **Admin Dashboard**: Centralized interface to view, add, update, and delete child nutrition records.
- **BMI Calculation**: Automatic computation of BMI based on height and weight.
- **Nutritional Classification**: Classifies childrenâ€™s nutritional status (underweight, normal, overweight, obese).
- **Data Persistence**: Records are stored locally using `localStorage`.
- **Alerts & Metrics**: Highlights abnormal nutrition statuses for immediate attention.
- **Responsive Design**: Works on desktop, tablet, and mobile devices.
